Movie App API

App to allow users to search Movies by a Title, Genre, Director and allow users to add favorite movies

Request URL HTTP Method Request Data Response Data
All Movie Titles http://localhost:8080/movies GET None Returns a list of all Movies.
Movie by Title http://localhost:8080/movies/Barbie GET None Returns Barbie movie information
Movie Genre http://localhost:8080/movies/genres/Actiom GET None A JSON object holding data about Action genre and a description of this genre.
Director by Name http://localhost:8080/movies/directors/Greta%20Gerwig GET None A JSON object holding data about Greta Gerwig director containing Name, short Bio and Birth Year.
List Users http://localhost:8080/users GET None Returns all Users
List Users by Username http://localhost:8080/users/Anne GET None Returns specific User
Update User Information http://localhost:8080/users/Matt PUT A JSON object with updates being made to User A JSON object holding data with users updated Name, Password, Email, Birthday and Favorite Movie.
Add User http://localhost:8080/users POST A JSON object containing Username, Password, Email, Birthday A JSON object holding data with User, Password, Email, Birthday, Favorite Moive and Unique id.
Add Favorite Movie to User http://localhost:8080/users/Matt/movies/64ecfde3f11c5b367531faed POST None adds a movie by MovieID.
Remove Favorite Movie from User http://localhost:8080/users/Matt/movies/64ecfde3f11c5b367531faed DELETE None null.
Delete User http://localhost:8080/users/Beto DELETE None A message indicating User was removed.
