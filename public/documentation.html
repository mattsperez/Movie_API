Movie App API

App to allow users to search Movies by a Title, Genre, Director and allow users to add favorite movies

Request URL HTTP Method Request body data format Response body data format
list of all movie titles http://localhost:8080/movies GET None A JSON object holding data about all the movies.
list movie by title http://localhost:8080/movies/Nacho%20Libre GET None A JSON object holding data about a movie by title containing Title, Year, Rating, Description, Comments, Genre, Director, ImageURL.
list movie genre http://localhost:8080/movies/genre/Comedy GET None A JSON object holding data about a movie genre and a description of the genre.
list director http://localhost:8080/movies/directors/Jon%20Favreau GET None A JSON object holding data about a movie director containing Name, short Bio and Birth Year.
update user name http://localhost:8080/users/1 PUT A JSON object containing users updated name A JSON object holding data with users updated Name, id, and Favorite Movie.
add user and create unique id http://localhost:8080/users POST A JSON object containing a user name A JSON object holding data with user and unique id.
add users favorite movie http://localhost:8080/users/1/Hot%20Fuzz POST A JSON object containing a users name and favorite movie A message indicating a movie title was added to users array.
delete users favorite movie http://localhost:8080/users/2/Elf DELETE None A message indicating a movie title was removed from users array.
delete user from array http://localhost:8080/users/2 DELETE None A message indicating a user was removed from array.
